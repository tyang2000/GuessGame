;	Source Code
;*************************************************************************

The source code is under "GuessGame" directory.

The application is developed in IntelliJ IDE with Maven environment. You
can build the unit test application by typing the following.

    mvn package.

The code was developed in Windows environment.
	
Note that GuessGame.properties is not used. The application currently use
system parameters to define the upper and lower bound numbers (the range).
